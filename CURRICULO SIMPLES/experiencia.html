Habita Recife

Desenvolvimento em grupo de uma plataforma fullstack em Java com Spring e Javascript com React.

A Plataforma Habita Recife é uma solução abrangente para administração de condomínios,
oferecendo ferramentas para gestão de moradores, controle financeiro, comunicação,
tarefas administrativas e manutenção.
